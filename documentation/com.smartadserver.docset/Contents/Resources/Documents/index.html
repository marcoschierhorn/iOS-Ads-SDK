    Installing this documentation in XCode

    If you want to install this Documentaton in Xcode, for quick reference, copy the com.smartadserver.docset in :

    • /Applications/Xcode.app/Contents/Developer/Documentation/DocSets for Mountain Lion and Lion when using Xcode 4.3 or superior
    • /Developer/Documentation/DocSets for Lion and Snow Leopard when using Xcode 4.2 or inferior
    • /Users/yourlogin/Library/Developer/Shared/Documentation/DocSets/ for Leopard
