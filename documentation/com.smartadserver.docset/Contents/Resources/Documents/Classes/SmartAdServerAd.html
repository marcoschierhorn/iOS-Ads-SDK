Conforms to NSCoding
NSCopying
Declared in SmartAdServerAd.h

Overview

A SmartAdServerAd object represent the ad data, as it has been programmed in the Smart AdServer interface You can create one of those objects, if you need to display an offline ad :

SmartAdServerAd * ad = [[SmartAdServerAd alloc] init];
ad.creativeURL = portraitUrl;
ad.creativeLandscapeUrl = landscapeUrl;
[banner displayThisAd:ad];
